Help

This script is optimized to remove all AIO tweaks by restoring files from their original backups
and restores image files to the default Red color scheme.

This script will safely restore your system if all modified files have a corresponding backup of the original file in same location with added '.org' extension, AIO does this automatically before modifying each file for the first time.
If you have modified any important files yourself I assume you know what you are doing.

Although it is strongly recommended that backups files are kept since they are not a detriment to the system, the option to delete all backups is provided to use at your own risk.
Files that will not be reverted:
For safety, AIO does not re-enable watchdog nor revert the files sm.conf and opera.ini. This is to prevent your system from ending up in a bootloop.
*Autorun & Recovery scripts are not removed to ensure access is not lost if firmware version is currently on or updated to v59.00.502+. Removing autorun scripts from v59.00.502+ may result in a full lockout of access from your CMU including tweak installs.
